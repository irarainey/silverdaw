# Application resources

Static files referenced at runtime by the Electron main process and at
packaging time by electron-builder. Kept **outside** the build output
so they're addressable via `app.getAppPath()` in `pnpm dev` and via
`process.resourcesPath` in a packaged install, without needing to be
re-emitted by Vite.

## `icons/`

Generated by [`scripts/Build-InstallerArt.py`](../../scripts/Build-InstallerArt.py)
from `frontend/resources/icons/256x256.png` (the source Silverdaw logo).

| File                                                              | Purpose                                                                                                                                                                                                                  |
| ----------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `icon.ico`                                                        | Windows window / taskbar / alt-tab icon. Wired in `frontend/src/main/index.ts` via `join(app.getAppPath(), 'resources', 'icons', 'icon.ico')`. Defensive lookup: if missing, Electron falls back to its default icon and `main.log` notes the absence — the app still starts. |
| `silverdaw-file.ico`                                              | File-type icon for `.silverdaw` documents (white page with folded corner + Silverdaw logo). Registered by electron-builder via `fileAssociations` in `electron-builder.yml`. |
| `16x16.png` … `256x256.png`                                       | Per-size PNGs. `256x256.png` is also the input to the installer-art generator script. Picked up by electron-builder when packaging.                                       |

## NSIS installer banners (in `frontend/resources/`)

| File                       | Purpose                                                                  |
| -------------------------- | ------------------------------------------------------------------------ |
| `installerHeader.bmp`      | 150×57 — the small banner at the top of each wizard page (logo on white). |
| `installerSidebar.bmp`     | 164×314 — the welcome-page sidebar (logo on black-to-grey gradient).      |
| `uninstallerSidebar.bmp`   | 164×314 — same artwork on the uninstaller's welcome page.                |

All three are auto-detected by electron-builder from `directories.buildResources`
(which is `frontend/resources/`). Regenerate them along with the file icon by running:

```powershell
pip install Pillow
python scripts/Build-InstallerArt.py
```

The outputs are checked into git so the normal release build doesn't need
Python on the PATH.
